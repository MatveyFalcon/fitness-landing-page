# Fitness Landing Page

Учебный проект — верстка лендинга фитнес-клуба по макету из Figma.

## 📌 О проекте

Проект создан для практики HTML и CSS.  
В рамках работы реализована адаптивная верстка лендинга с использованием современных подходов.

## 🚀 Что реализовано

- Семантическая HTML-разметка
- Flexbox / Grid
- Pixel Perfect (по макету Figma)
- Чистый и структурированный CSS

## 🛠️ Технологии

- HTML5
- CSS3
- Flexbox / Grid
